# src/app/(site)/business-lunch/compare/page.tsx

Страница сравнения бизнес-ланчей. URL: /business-lunch/compare?ids=1,2,3,4. Серверный компонент: загружает данные по IDs из БД (до 4 ланчей). Показывает сравнительную таблицу с подсветкой лучших/худших значений: цена (дешевле/дороже), расстояние (ближе/дальше), рейтинг (выше/ниже), время работы, дни недели, статус 'сейчас подают', адрес, позиции меню. Горизонтальная прокрутка для 3-4 ланчей.
