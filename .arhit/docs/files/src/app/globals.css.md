# src/app/globals.css

Главный CSS-файл приложения, импортирующий Tailwind 4 через @import 'tailwindcss'. Содержит design tokens в @theme (color/surface/fg/border/radius/font/animation), базовые стили в @layer base, utility-классы в @layer utilities, keyframes, View Transitions API правила и универсальный prefers-reduced-motion. Animation-секция переписана под ANIMATIONS_GUIDE: удалены hero-float/hero-card/site-route-enter/vt-card-launch/data-vt-active, добавлены --dur-*/--ease-*/--animate-* токены, глобальное press-feedback для button/a/[role=button] с :active scale(0.97), .shadow-hover utility через ::after для hover-теней без анимации box-shadow, 4 keyframes (fade-in/pop-in/hero-enter/skeleton-shimmer), @view-transition{navigation:auto} для автоматического VT cross-fade, универсальный prefers-reduced-motion. Файл https://github.com/lunchhunter/lunchhunter/blob/main/src/app/globals.css
