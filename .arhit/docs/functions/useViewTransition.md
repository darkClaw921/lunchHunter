# useViewTransition

Обёртка над document.startViewTransition для НЕ-навигационных state-переходов (theme switch, accordion expand/collapse, tab switch). Сигнатура: () => (callback: () => void) => void. Внутри возвращённой функции: если typeof document === 'undefined' или !document.startViewTransition → callback() напрямую (SSR-safe + graceful fallback для Telegram Mini App/старых WebView); иначе document.startViewTransition(() => flushSync(callback)). flushSync обязателен — он форсирует синхронный flush React updates внутри VT-коллбэка, без него React может отложить state update (concurrent mode) и VT API снимет snapshot до применения нового состояния. Для НАВИГАЦИОННЫХ переходов этот хук НЕ нужен — работает @view-transition { navigation: auto } в CSS + startTransition в navigate(). Файл: src/lib/hooks/useViewTransition.ts. Паттерн из гайда §6.3.
